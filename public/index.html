⚠️ Signal не підключається — перевірте Docker
Підключення триває понад хвилину. Найчастіша причина — Docker Desktop не працює або контейнери Signal зупинені.
  1. Переконайтесь, що Docker Desktop запущений (іконка в системному треї без червоного знака).
  2. Якщо Docker завис — правий клік на іконці Docker Desktop → Restart. Зачекайте ~30 секунд.
  3. Після перезапуску Docker відкрийте термінал і запустіть контейнери Signal:
cd D:\Armor\whatsapp-bot docker compose -f docker-compose.signal.yml up -d
  1. Натисніть «Підключити акаунт» у картці Signal вище.
  2. Якщо Signal раніше не реєструвався — натисніть «Видалити сесію», щоб отримати QR-код для сканування.

Що тут відбувається?

Інтеграція — підключення номера через QR (як WhatsApp Web), перегляд стану та параметрів сесії.

Автоматизації — напрямок задається як «Звідки → Куди»: WhatsApp/Signal → WhatsApp/Signal/FastAPI; фільтри за текстом і частотами (при двох полях — логіка АБО).

  • Інші напрямки (WA→WA, Signal) — пізніші етапи.
  • Окремий токен * у полі — пропуск усіх повідомлень без фільтрації по цьому автоматизації.
